# 아주 보통의 하루 - Aboha

평범한 일상의 회복을 위한 프로젝트, Aboha

💡 Aboha는 **소확행(소소하지만 확실한 행복)**을 추구하는 사용자들에게 일상의 만족을 제공하기 위해 개발된 애플리케이션입니다.
